# Support

Paige is in public alpha.

- Use the [`setup` skill](./.agents/skills/setup/SKILL.md) for installation and
  readiness checks.
- Open a focused bug report when documented behavior fails reproducibly.
- Open a feature request when a missing capability would generalize beyond one
  repository or provider conversation.
- Use [private vulnerability reporting](https://github.com/peelar/paige/security/advisories/new)
  for security concerns.

There is no paid support or guaranteed response time. Please remove secrets,
tokens, private repository contents, and sensitive provider context from public
issues.
